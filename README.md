# Choice-Driven Fictional Travel Agency

A full-stack web application designed to generate dynamic travel narratives based on user-inputted psychological parameters (mood, personality traits, and preferences).

### 🛠️ Tech Stack & Architecture
- **Backend:** Python, Flask Microframework
- **Frontend:** HTML5, CSS3, JavaScript
- **Data Persistence:** Server-side custom File Handling (`.txt` / `.json` storage tracking)

### 💻 Key Software Engineering Implementation
- **Strict OOP Design:** Utilized Encapsulation to isolate user profile modules and Composition to map choice vectors to narrative blocks dynamically.
- **Production Path Management:** Implemented dynamic absolute path routing (`os.path.abspath`) to maintain clean cloud filesystem execution on PythonAnywhere.
- **Exception Handling:** Engineered robust validation blocks to intercept file I/O operations and malformed network inputs cleanly.
